#### Запустить сервер
1. `rails server`

2. `rails s`

#### Миграция БД
`rake db:migrate`

#### Отменить последнюю миграцию
`rake db:rollback`

#### Убрать все миграции
`rake db:reset`

#### Все версии руби
`ruby -v`

#### Установить версию ruby
`rbennv install 2.58`

#### Изменить версию ruby
`rbennv shell 2.5.8`

#### Все версии руби
`rbennv install --list`
