### Убить процесс
1. Получить все процессы

    `ps aux`
    
2. Убить процесс

    `kill -9 28691`, 28691 - номер процесса
    
### Открыть hosts файл
`sudo nano /etc/hosts`


### Получить статус приложения
`service postgresql status`


### Старт приложения
`service postgresql start`

### Рестарт приложения
`service postgresql restart`

### Стоп приложения
`service postgresql stop`
